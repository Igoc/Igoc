## Simple Memory Manager

<div align="center">
    <img src="https://github.com/Igoc/Igoc/blob/main/Image/Simple%20Memory%20Manager.png?raw=true"/>
</div>

**설명** <br/>
사전에 작성된 메모리 할당과 해제에 대한 시나리오를 시각화해주는 시뮬레이션 프로그램 <br/><br/>

**일정** <br/>
2017.09 ~ 2017.09 <br/><br/>

**기술 스택** <br/>
C++, C++/CLI, C#, Windows Forms <br/><br/>

**기여도** <br/>
개인 프로젝트 (100%) <br/><br/>

**세부 기능** <br/>
- C++로 핵심 로직을 구현하여 라이브러리 파일로 만들고, 해당 라이브러리 파일을 C++/CLI를 통해 C#과 연동
- 운영체제의 메모리 할당 전략 가운데 First-Fit과 Best-Fit을 이용하여 메모리 할당 시나리오 구현
- 시나리오 자동 재생 모드와 수동 재생 모드 지원

## Search

<div align="center">
    <img src="https://github.com/Igoc/Igoc/blob/main/Image/Search.png?raw=true"/>
</div>

**설명** <br/>
스마트폰 카메라로 일러스트레이터의 명함을 촬영하면, 해당 일러스트레이터에게 연락할 수 있는 방법을 제공하는 프로그레시브 웹 애플리케이션 <br/><br/>

**일정** <br/>
2018.07 ~ 2019.06 <br/><br/>

**기술 스택** <br/>
Vue.js, Python, Flask, Scikit-learn, Keras, MariaDB <br/><br/>

**기여도** <br/>
개인 프로젝트 (100%) <br/><br/>

**세부 기능** <br/>
- Vue.js에 PWA (Progressive Web Apps) 기술을 적용
- HTML5의 카메라를 이용하여 스마트폰에서 사진을 캡처
- 샴 네트워크를 이용하여 촬영된 사진의 특징 벡터를 추출하고, 랜덤 포레스트를 통해 유사한 일러스트레이터 명함 매칭
